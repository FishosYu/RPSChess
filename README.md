# RPSChess

单人可玩的人机小游戏：石头剪刀布版回合制简易象棋！规则简单易上手，3分钟之内就能完成一局。

## Run

Double-click `index.html` to play.

The page uses a local browser bundle plus copied audio files:

- `rpschess.bundle.js`
- `music.mp3`
- `sfx/button-click.mp3`
- `sfx/broke.mp3`
- `sfx/fight.mp3`
- `sfx/win.mp3`
- `sfx/lose.mp3`

The copied source also keeps the original `game/`, `packages/`, `i18n/`, and `public/` layout for future development. Piece visuals are drawn dynamically by the game code.
